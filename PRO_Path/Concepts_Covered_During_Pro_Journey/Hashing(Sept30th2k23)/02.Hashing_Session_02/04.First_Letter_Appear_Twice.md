# Can use SET's to find 1st Letter that occur twice

## Brute Force

```cpp
class Solution {
public:
    char repeatedCharacter(string s) {
        unordered_map<char,int> mp;

        for(int i=0;i<s.size();++i)
        {
            char ch=s[i];
            if(mp.find(ch)==mp.end()) mp[ch]++;
            else
            {
                return ch;
            }
        }
        return s[0];
    }
};
```
